## Understanding Tool Information

### Tool Cards

Each tool card provides essential information at a glance:

- **Image**: Visual representation of the tool
- **Name**: Official tool name
- **Category Badge**: Primary classification
- **Description**: Brief overview (3 lines max)
- **Tags**: Up to 3 tags shown (+ count if more)
- **Date**: Release or discovery date
- **Source**: Where we found the tool

### Tags Explained

Tags help you quickly identify tool capabilities:
- **Functional Tags**: What the tool does (e.g., "image", "coding", "writing")
- **Use Case Tags**: How it's used (e.g., "marketing", "design", "productivity")
- **Technology Tags**: What it's built with (e.g., "AI", "ML", "GPT")

### Sources

Tools are discovered from various sources:
- **ProductHunt**: Popular product launch platform
- **TechCrunch**: Leading tech news outlet
- **The Verge**: Technology news and media network
- **VentureBeat**: Technology news site
- **Reddit**: AI and machine learning communities
- **AI Directories**: Specialized AI tool directories
